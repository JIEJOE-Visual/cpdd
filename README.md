## 🛠 Set up
1. 整个网页项目使用的是原生WEB技术，没有借助任何开发框架，所以，你可以直接点击index.html进入网页
2. 但是，JIEJOE还是建议你用IDEA工具例如VSCode打开，否则网页可能fecth不到音频文件，包括出一些小的BUG
3. 整个工程文件因为比较简单，所以JIEJOE没有添加什么注释，相信大家都基本可以看懂👍
